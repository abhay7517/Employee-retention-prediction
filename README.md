# Employee Retention Prediction (Machine Learning Project)

## 📌 Project Overview
This project predicts whether an employee is likely to leave the company using machine learning techniques.

## 🎯 Objective
To help organizations identify employees at risk of leaving and take preventive actions.

## 📊 Dataset
- HR dataset with employee details like experience, education, company type, etc.
- Target Variable: 1 = Looking for job change, 0 = Not looking

## ⚙️ Steps Performed
- Data Cleaning (handled missing values)
- Exploratory Data Analysis (EDA)
- Feature Engineering
- Encoding categorical variables
- Feature Scaling

## 🤖 Machine Learning Models Used
- Logistic Regression
- Random Forest
- XGBoost (Best Model)

## 📈 Model Performance
- Accuracy: ~85%
- Evaluated using ROC-AUC and Confusion Matrix

## 🛠 Tools & Technologies
- Python
- Pandas, NumPy
- Scikit-learn
- XGBoost
- Matplotlib, Seaborn

## 🚀 Outcome
This model helps companies reduce employee attrition by predicting risk in advance.
